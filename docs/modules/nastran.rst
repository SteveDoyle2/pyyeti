Tools for working with Nastran data
===================================

:doc:`nastran <nastran>` is a subpackage of pyYeti that imports
functions from the modules: :doc:`bulk <nastran/bulk>`, :doc:`n2p
<nastran/n2p>`, :doc:`op2 <nastran/op2>`. It also imports the module
:doc:`op4 <nastran/op4>`.
