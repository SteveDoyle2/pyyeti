Tools for working with Nastran data
===================================

:doc:`nastran <nastran>` is a subpackage of pyYeti that imports
functions from the modules: :doc:`bulk <bulk>`, :doc:`n2p <n2p>`,
:doc:`op2 <op2>`. It also imports the module :doc:`op4 <op4>`.
